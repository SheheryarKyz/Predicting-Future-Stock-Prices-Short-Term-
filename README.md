# Stock Price Prediction

## Objective
Predict the next day's closing stock price using historical stock market data.

## Dataset
Yahoo Finance (AAPL Stock Data)

## Tools Used
- Python
- Pandas
- NumPy
- yFinance
- Scikit-Learn
- Matplotlib
- Seaborn

## Model
Linear Regression

## Features
- Open
- High
- Low
- Volume

## Target
- Next Day Closing Price

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Results
The Linear Regression model successfully learned relationships between stock market indicators and future closing prices. Predictions closely followed actual stock price movements.
